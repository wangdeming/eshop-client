# eshop-client

旅游电商客户端
# 安装依赖
npm install // or yarn install

# 如何运行
npm run dev
浏览器输入localhost:8080

# 如何部署
npm run build
将根目录dist目录下的文件全部放置在静态web服务器上

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
